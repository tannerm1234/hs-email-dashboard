DEPLOYMENT STEPS:
1. Copy ALL files to your GitHub repository
2. In Vercel, import the repository
3. Add environment variables in Vercel:
   - HUBSPOT_TOKEN
   - HUBSPOT_PORTAL_ID
   - MAX_WORKFLOWS (optional)
   - API_DELAY_MS (optional)
4. Deploy!
5. Test at: your-app.vercel.app/debug
